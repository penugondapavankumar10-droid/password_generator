## Why shuffle is important?
-random.choice() - to randomly select letters, symbols, and numbers
-random.shuffle() - to shuffle the password characters for better security
-If a password is generated in a fixed sequence like letters → symbols → numbers, it becomes predictable and easier for hackers to guess.
So, I used shuffle to mix all characters randomly, making the password much stronger and more secure.
-Example output: *Z1nn&!3C
