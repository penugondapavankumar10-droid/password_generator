# 🔐 Password Generator (Python)

## 📌 Description

This is a simple Python-based Password Generator that creates a random and secure password based on user input.

The user can specify:

* Number of letters
* Number of symbols
* Number of numbers

The program then generates a shuffled password using these inputs.

---

## 🚀 Features

* Generates strong random passwords
* User-controlled password length
* Uses letters (A–Z, a–z), numbers (0–9), and symbols
* Randomly shuffles characters for better security

---

## 🛠️ Technologies Used

* Python
* random module

---

## ▶️ How to Run

1. Make sure Python is installed on your system
2. Open terminal / command prompt
3. Navigate to the project folder
4. Run the program:

```
python Password_generator.py
```

Welcome to Password Generator
